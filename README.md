# Student Hub

Student Hub is a simple PC application made using Visual Studio.  
It is designed as a small toolbox to help students with common tasks.

---

## Features

The application includes the following tools:

• Calculator  
• Notes that can be written, saved, and loaded  
• Number conversion  
    • Binary  
    • Decimal  
    • Octal  
    • Hexadecimal  
• Unit conversion  
• Coin flip  

---

## Built With

• Visual Studio  (C#)  
• Desktop application for PC
